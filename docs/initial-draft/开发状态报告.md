## 一、项目概览

| 项目 | 内容 |
| --- | --- |
| 名称 | **FiveAvenues（五大道旅行文化指南）** / "Five Avenues Showing" |
| 定位 | 为天津五大道景区游客提供**个性化行程生成**与**社区评价分享**的一站式文旅指南网站 |
| 技术栈 | Node.js 原生 HTTP（无框架、**零 npm 依赖**，仅用内置模块 `http/fs/crypto/path/url`） |
| 运行环境 | Node.js >= 18 |
| 启动方式 | `cd app && npm start` → 访问 `http://localhost:3000` |
| 默认管理员 | `admin` / `admin12345`（可通过 `ADMIN_USER` / `ADMIN_PASS` 覆盖） |
| 来源 | `DESIGN.md` §1、`REQUIREMENTS.md` 产品定位 |
